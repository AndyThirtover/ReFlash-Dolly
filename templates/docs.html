Command URLS

Command Actions
trajectory_to 'position'/'speed' Move to 'position' at a maximum of 'speed', another can be issued at any point
set_state ['IDLE'|'RUN'] sets the motor drive to IDLE or RUN
current 'value' Set maximium motor current to 'value' in amps
cancel_trajectory NOP cancels a trajectory_to - the NOP is required

Internal Connections

Connection Function
BCM GPIO 5 Button 0 - All Off
BCM GPIO 6 Button 1 - Configured Function, default 'rotate'
BCM GPIO 18 NeoPixel Strip 1 Drive
